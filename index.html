LinDelInt

LinDelInt is an off-grid interpolator of 2D and 3D scalar and vectorial fields based on Delaunay tessellations.

Install it

conda install -c uibcdf lindelint

Use it

import lindelint as ldl
import numpy as np

points = np.array([[-4.5, -6.8],
                   [ 7.5, -5.3],
                   [-4.1,  5.1],
                   [ 5.7,  4.7],
                   [-6.6,  0.2],
                   [ 2.9, -0.7]])

properties = np.array([[ 0.5, -0.5],
                       [-0.7, -0.9],
                       [ 0.0,  0.8],
                       [-0.6,  0.9],
                       [ 0.4,  0.3],
                       [-0.3, -0.5]])

new_points = np.random.uniform(-10, 10, (400, 2))

interpolator = ldl.Interpolator(points, properties)

new_properties = interpolator.do_your_thing(new_points)
